# Restaurant Listing Project

This is a simple Zomato-style restaurant listing web page built using **HTML, CSS, and JavaScript**.

## Features
- Displays a list of restaurants with images, ratings, cuisine, and location.
- Live search: filter restaurants by name, cuisine, or location.
- Responsive and simple design.

## How to Use
1. Open `index.html` in your browser.
2. Use the search box to filter restaurants.
3. Enjoy browsing restaurants!
